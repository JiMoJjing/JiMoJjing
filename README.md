### 💬저에 대해서 ###
온라인 게임을 좋아하고 많이 플레이해 왔으며, 나아가 게임 개발자로의 길을 걷고 있는 **지용현** 입니다.
20년 이상의 플레이 경험을 바탕으로, 더 재미있고 완성도 높은 게임을 설계하고 구현하는 게임 개발자를 꿈꾸고 있습니다.
### 🏫학력/교육 ###
- 강릉원주대학교 전자공학과 전공 **2014.03 - 2022.02**
- 원티드 포텐업 게임개발자 양성과정 1기   **2024.12 - 2025.06**
