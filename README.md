### 💬저에 대해서 ###
온라인 게임을 좋아하고 많이 플레이해 왔으며, 나아가 게임 개발자로의 길을 걷고 있는 **지용현** 입니다.

## 프로젝트 ##
### 프로젝트 RTS (Robot ThridPerson Shooting) ###
로봇 캐릭터를 메인으로 한 삼인칭 슈팅 게임입니다.
### 프로젝트 OW (Overwatch) ###
오버워치에서 유명한 캐릭터 "겐지"를 삼인칭으로 구현한 프로젝트 입니다.
